# noths-frontend

## Prerequisites

* Node.js v4.3.0
* npm v3.7.1

## Installation

Install node module dependencies

`npm install`

Create webpack module mappings (must be re-run when modules change)

`npm run create-bundle-indexes`

## Run the server

`npm start`

Development mode with auto-reloading

`npm run dev`

## Tests

Run all the Tests 

`npm test`

Run the tests and watch for changes 

`npm test -- --watch`
